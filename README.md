# Вебзастосування "Список завдань"

Вебзастосування "Список завдань" дає змогу вести облік нових та виконаних завдань на одній сторінці.

Проєкт створено за допомогою Node.js та Express. Для відображення сторінки використовується шаблонізатор EJS, а для обробки даних форми використовується body-parser.

## Технології

- Node.js
- Express
- EJS
- Body-parser
- HTML
- CSS
- Git та GitHub

## Локальний запуск

Для запуску вебзастосування потрібно виконати такі команди:

```bash
git clone https://github.com/SanchezGr/list.git
cd list
npm install
node index.js
